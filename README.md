This is an add-on for Simhub, adding:
- Engine RPM (tested and working)
- Turn Signals (tested and working)
- Flags (not tested)
- Lights (not tested)
- Car Spotter (tested and working)
- Handbrake (not tested)

Other functions will be added in the future.

Download the files to the main Simhub folder.

How it works:
1. Set the IP address of the LED without the port.
2. Set the number of LEDs and the type.
3. Save.
4. Add the desired function.
5. Click "Edit LED" and select the LEDs to be assigned to the function.
IMPORTANT:
When selecting LED RPM, the plugin remembers the order in which the LEDs were clicked, and the LEDs will light up in that order.
